## Key Fields for Tax Return

| 1099-DIV Box | Flows To |
|-------------|----------|
| Box 1a | Form 1040, Line 3b (ordinary dividends) |
| Box 1b | Form 1040, Line 3a (qualified dividends — taxed at lower rate) |
| Box 2a | Schedule D (capital gain distributions) |
| Box 4 | Form 1040, Line 25b (withholding) |
| Box 7 | Form 1116 or Schedule A (foreign tax credit/deduction) |
| Box 12 | Form 1040, Line 2a (tax-exempt interest) |

## Triggers

- If total dividends from all 1099-DIVs > $1,500 → **Schedule B required**
- Box 1b > 0 → Use Qualified Dividends and Capital Gain Tax Worksheet for Line 16
- Box 2a > 0 → Report on Schedule D
