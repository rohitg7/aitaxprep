## Mortgage Interest Deduction Limits

```
Was the mortgage originated on or before December 15, 2017?
├── Yes → Interest is deductible on up to $1,000,000 of acquisition debt
│         ($500,000 if married filing separately)
└── No → Interest is deductible on up to $750,000 of acquisition debt
          ($375,000 if married filing separately)
```

> ⚠️ **VERIFY** these limits against [Publication 936](https://www.irs.gov/publications/p936) for the current tax year.

## Multiple Mortgages

- If the taxpayer has multiple 1098 forms, sum all Box 1 amounts for Schedule A Line 8a
- The mortgage interest deduction limit applies to the **combined** outstanding balance (Box 2) across all mortgages
- If total outstanding principal exceeds the limit, only a prorated portion of interest is deductible
- Use the **Home Mortgage Interest Deduction Worksheet** in Publication 936 if limits apply

## Triggers

- Box 1 > 0 → Consider **Schedule A** (itemized deductions)
- Box 2 exceeds acquisition debt limit → Prorate interest deduction using Publication 936 worksheet
- Box 5 > 0 → Mortgage insurance premiums may be deductible (check current year availability)
- Box 6 > 0 → Points are generally deductible in the year paid for a primary home purchase

## Important Notes

- Mortgage interest is only deductible if the taxpayer **itemizes** deductions on Schedule A
- Home equity loan interest is only deductible if the loan was used to buy, build, or substantially improve the home
- If Box 7 is "No" (property address differs from borrower address), the property may be a second home or rental — different rules apply
- **IRS Verification**: [Publication 936](https://www.irs.gov/publications/p936) — Home Mortgage Interest Deduction
