## Key Fields for Tax Return

| 1099-R Box | Flows To | Condition |
|-----------|----------|-----------|
| Box 1 | Form 1040, Line 4a (IRA) or Line 5a (pension/annuity) | Depends on Box 7 IRA checkbox |

| Box 3 | Schedule D (capital gain portion) | If applicable |

## Determining IRA vs. Pension Reporting

```
Is Box 7 IRA/SEP/SIMPLE checkbox marked?
├── Yes → Report on Form 1040 Lines 4a/4b (IRA distributions)
│   └── May need Form 8606 if non-deductible contributions
└── No → Report on Form 1040 Lines 5a/5b (Pensions and annuities)
    └── May need to use Simplified Method worksheet
```

## Triggers

- Code 1 or S in Box 7 → **Form 5329** (additional tax on early distributions)
- Code G → Rollover, verify no taxable amount
- IRA checkbox + non-deductible contributions → **Form 8606**
- Multiple 1099-Rs → Sum all Box 1 for Line 4a or 5a, sum all Box 2a for Line 4b or 5b
