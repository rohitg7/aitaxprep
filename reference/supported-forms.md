## Not Supported

- **State tax returns** — FFFF only supports federal filing
- **Form 1040-SR** — Not available on FFFF; use Form 1040 instead
- **Form 1040-ES** — Print-only on FFFF, not e-filed
- **Form 1040-V** — Print-only payment voucher
- **Amended returns (1040-X)** — Not supported at this time

## Requesting New Form Support

If you need a form that's available on FFFF but not yet mapped, please open an issue on our GitHub repository.
